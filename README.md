# Mastermind
Console implementation of Mastermind game with numbers instead of colors. Range of numbers and number of rounds easy to change in code.

No system("cls"). Escape codes used to manipulate console carriage.

Controls:
- number keys 1-9
- left/right arrow
- backspace
- enter
